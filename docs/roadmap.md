# LightClaw 开发路线图

## 当前状态 (v0.1.0)

已完成的 MVP 功能：

- [x] 项目框架搭建
- [x] Agent Runtime 实现
- [x] 工具系统 (12 个工具)
- [x] Gateway 日志记录
- [x] DataPool 数据构建
- [x] 评测框架
- [x] 训练数据导出接口
- [x] 前端 Dashboard
- [x] 内置任务集 (20 个)
- [x] 完整文档

## 技术债务

当前已知需要改进的部分：

1. **类型注解**: 部分函数缺少完整的类型注解
2. **测试覆盖**: 测试用例较少
3. **错误处理**: 部分错误处理不够完善
4. **文档**: API 文档可以更详细
5. **性能**: 部分查询可以优化
