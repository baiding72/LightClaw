# 示例任务配置

这个目录包含用于演示的示例任务配置文件。

## 可用任务

### 信息整理类
- `info_001`: 从网页中提取会议信息，写入笔记
- `info_002`: 从网页中提取招聘信息，生成总结

### 待办/日程管理类
- `todo_001`: 创建一个简单的待办事项
- `todo_002`: 创建一个日历事件

### 网页表单交互类
- `form_001`: 在本地表单页面填写信息并提交

### 跨工具多步任务类
- `multi_001`: 搜索活动页面，创建日历事件，写提醒笔记
- `multi_002`: 读取本地文件中的待办列表，导入系统

## 使用方法

1. 启动本地演示页面
2. 运行 Agent 执行任务
3. 查看执行结果和轨迹

## 文件说明

- `meeting_info.txt`: 会议信息文件，用于信息提取任务
- `meeting_notes.md`: 待办事项列表，用于导入任务
